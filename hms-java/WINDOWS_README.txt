WINDOWS QUICK START - Java Spring Boot
=======================================

REQUIREMENTS:
1. JDK 17+  ->  https://adoptium.net/
2. Maven 3.8+  ->  https://maven.apache.org/download.cgi

STEPS:
1. Unzip this folder
2. Double-click START.bat
3. Wait for build (first time: 3-5 minutes)
4. Open http://localhost:8080
5. Login: admin / admin123

DO NOT close the black window while using the app.

IF JAVA NOT FOUND:
- Install JDK 17 from https://adoptium.net/
- Check "Add to PATH" during install
- Restart PC

IF MAVEN NOT FOUND:
- Install Maven OR ensure mvnw.cmd exists in this folder

RESET DATABASE:
- Delete the "data" folder
- Run START.bat again
