WINDOWS QUICK START
===================

1. Install Python 3.10+ from https://www.python.org/downloads/
   IMPORTANT: Check "Add python.exe to PATH" during install!

2. Unzip this folder anywhere (e.g. Desktop\GrandStay-HMS)

3. Double-click:  START.bat

4. Login at http://127.0.0.1:5000
   Username: admin
   Password: admin123

5. DO NOT close the black window while using the app.


IF NOT WORKING:
---------------
A) Run TEST_SETUP.bat  (shows the exact error)
B) Run RESET_DATABASE.bat
C) Run START.bat again

Still broken? Send a screenshot of the black window error text.
